Booking coffee shops for remote workers

• This mini project was a part of design sprint where with the limited time, a prototype was supposed to be designed with all the basic and important functionalities . The project was to design an app to help the remote workers to find great coffee shops and public places to work from.

• Tool that was used to design this app was Marvel

• As this was a design sprint , this project was done based on sprint process where, Day 1 was to understand the problem, Day 2 was sketching using lighting demo method and sketching the possible solutions of the most critical screen using the Crazy 8s method. Day 3 was to decide on the final sketch and prepare a story board. Day 4 was to prototype the most used functionality quickly using the design tool and finally Day 5 was testing with the participants.

• As a result a prototype was designed with all the basic functionalities where a user was able to book a coffee shop using this app.
